# architectomics-benchmarks

Open benchmarks, legacy DOSA fixtures, and evaluation tasks for Architectomics.

## Status

This repository is the initial bootstrap scaffold.

It is currently staged under the `agourakis82` account and is intended to move into the `Architectomics` GitHub organization once that organization exists.

## Purpose

`architectomics-benchmarks` is the public evaluation layer of the program.

It is responsible for:

- frozen legacy DOSA fixtures
- open benchmark tasks for platform and model evaluation
- baseline outputs and scoring references
- reproducible benchmark reports

## Planned Layout

- `legacy-fixtures/` frozen DOSA-compatible reference material
- `tasks/` benchmark task definitions
- `baselines/` baseline outputs and scoring references

## Next Steps

1. Freeze a representative DOSA legacy subset.
2. Define the first topology-aware benchmark tasks.
3. Publish public baseline expectations.
